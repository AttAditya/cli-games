# Hangman game

## Play

```bash
$ python hangman.py
```

or 

```bash
$ python hangman.py [language]
```

#### Example

```bash
$ python hangman.py pt-br
```

## Node.js 

```bash
$ node index.js
```

or 

```bash
$ node index.js [language]
```

#### Example

```bash
$ node index.js pt-br
```
